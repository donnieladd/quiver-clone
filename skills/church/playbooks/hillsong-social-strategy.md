# Playbook: Hillsong Social Strategy (reference)

**Source:** Hillsong social media strategy document shared via [Open Network resource 4431](https://open.life.church/resources/4431-church-social-media-strategy-documents) alongside Church of the Highlands scheduling doc.

## What this playbook represents

Not a client profile — a **reference playbook** for social **strategy framework** (pillars, tone, planning rhythm). Pair with Highlands scheduling grid skill.

**Skill:** `church.social.apply_hillsong_strategy_framework`  
**Companion:** `church.social.use_highlands_scheduling_grid`

## Framework (adapt to any profile)

1. **Define social purpose** — Why does this church exist on social? (invite, disciple, celebrate, resource)
2. **Audience segments** — Members, seekers, leaders, alumni/creatives
3. **Content buckets** — Teaching, worship, community, serve, events, BTS
4. **Platform roles** — IG for culture, YT for long-form, FB for events/local, etc.
5. **Planning rhythm** — Weekly batch + monthly pillar audit
6. **Voice guardrails** — Align to profile `voice.do` / `voice.dont`

## When to use

- Setting up a new church client's social system
- Auditing an existing grid that's pastor-heavy or event-heavy
- Before `church.story.plan_story_content_calendar`

## Steal / avoid

| Steal | Avoid |
|-------|-------|
| Written strategy doc before templates | Templates without strategy |
| Bucket balance reviews | Random posting |
| Platform-specific adaptation | Identical cross-post |
