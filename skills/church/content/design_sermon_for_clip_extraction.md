# Design sermon for clip extraction

## Purpose

Plan sermon **before Sunday** so 5–15 short clips exist by design — Mike Todd: people remember **pictures** more than words.

## Preach planning checklist

1. Mark 3–5 **visual illustration** moments (props, movement, demonstration).
2. Write **7-word max** punch lines for point slides (clip-friendly).
3. Note timestamp targets for team: intro hook, each point peak, altar moment.
4. Frame illustrations for **9:16 center-safe** crop.
5. Plan thumbnail face/emotion at strongest visual beat.

## Outputs for production team

| Marker | Platform |
|--------|----------|
| Hook (0:00–0:30) | Reels, TikTok |
| Point clips | YouTube Shorts, IG |
| Full illustration | YouTube long-form chapter |

## QA checks

- [ ] At least 3 clip moments marked in outline
- [ ] Each moment works without sermon context (standalone test)
