# Write mission and vision in brand guide

## Purpose

Anchor all church media in stated mission, vision, and values so volunteers make consistent tone choices.

## When to use

Creating or refreshing the church brand guide.

## When NOT to use

Skip when a higher-level workflow has not locked brand/series prerequisites, or when approval gates in `workflows/approval-governance.md` are not satisfied.

## Inputs

- `mission_statement`
- `vision_statement`
- `core_values_list`

## Decision rules

- Follow brand guide and series kit when present.
- Prefer template duplication over new layout.
- Escalate theology-sensitive copy to pastoral review.

## Steps

1. Paste approved mission/vision from leadership (do not paraphrase without sign-off).
2. List 3–7 values with one-sentence behavioral definitions each.
3. Add 'what we sound like' examples for social and stage.
4. Publish to brand hub and Canva kit description field.
