# Run brand education session for staff and volunteers

## Purpose

Mirror Faith Chapel's **Brand Education Session** model (PDF + MP3 on [brand hub](https://www.faithchapel.net/brand)) for staff/volunteer alignment.

## Agenda (60 min)

1. Why brand matters (movement, not clipart)
2. Walk through Quick Reference Guide
3. Logo clear space + minimum sizes
4. Template duplication demo — **`church.ops.duplicate_template_not_redesign`**
5. Photography standards
6. Q&A — record for archive

## Deliverables

- Updated one-page quick guide if gaps found
- Attendance + sign-off for volunteer leads
