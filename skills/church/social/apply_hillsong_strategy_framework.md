# Apply Hillsong social strategy framework

## Purpose

Use **`playbooks/hillsong-social-strategy.md`** as a **reference lens** — platform-native rhythm, worship + message + community mix — adapted to each profile's voice (not Hillsong copy).

## Framework pillars (adapted)

1. **Platform-native** — format matches channel (Reels vs feed vs Stories)
2. **Rhythm** — predictable weekly beats, not random posts
3. **Worship + word + witness** — three content types in balance
4. **Global church, local expression** — multi-campus/local when applicable

## Steps

1. Read Hillsong playbook (reference only).
2. Map pillars to profile `content_pillars` and pastor-face caps.
3. Output gap list vs current calendar.
4. Do **not** copy Hillsong visual identity — strategy only.

## When to use

- Quarterly social audit
- New social lead onboarding
- Before series launch social plan
