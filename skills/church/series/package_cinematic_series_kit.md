# Package cinematic series kit

## Purpose

Transformation Church–grade series drop: **iconic, slow-expiring** visual IP across stage, motion, social, YouTube.

## Kit manifest (minimum)

| Folder | Assets |
|--------|--------|
| `/KeyArt` | Master PNG/PSD, title SVG, variants |
| `/Motion` | Loops, opener 15–30s, stinger, lower thirds |
| `/Stage` | IMAG stills, environmental |
| `/Social` | Launch, week N, reel covers, thumbnails |
| `/Web` | Hero 2400×1200, OG 1200×630 |
| `/Docs` | Color, type, vision phrases, clip hooks |

## Decision rules

- Series look locked **week 1** — no mid-series palette drift
- Thumbnails tested at 1280×720 and small mobile preview
- Motion tested on sanctuary projector + YouTube still frame
