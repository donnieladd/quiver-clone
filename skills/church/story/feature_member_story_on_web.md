# Feature member story on web

## Purpose

Turn approved testimony into a **web feature** (not only a single social graphic) — community as editorial content.

## Steps

1. Structure: headline (benefit-led) → member quote → 2–3 short paragraphs → CTA (groups, visit, connect).
2. Hero image: art-directed crop; scrim if text overlay.
3. Generate OG companion: `church.web.add_open_graph_meta_companion`.
4. Tease on social with `church.social.template_community_story`.
5. Union: ensure seeker-first copy per profile `voice.seeker_first`.

## Export specs

- Hero: 2400×1200 sRGB
- OG: 1200×630
