# Plan story content calendar

## Purpose

Build a **30-day story-first calendar** before individual assets — community + values + series, not reactive event art.

## Steps

1. **`church.story.route_by_profile`**
2. Block recurring: sermon social, worship, groups, serve (1 each per 2 weeks min)
3. Add 2 community stories / month minimum
4. Add 1 values or campaign arc (`build_community_campaign_not_event`)
5. Project pastor-face ratio vs `story.pastor_face_ratio_max_30d`
6. Profile-specific: Highlands → `balance_four_mission_pillars`; TC → vision glossary pass
