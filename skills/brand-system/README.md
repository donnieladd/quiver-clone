# Brand System Skills Pack

Horizontal **design system** layer for Brand Studio — tokens, logos, export, lint.

- **Schema:** `schema.json`
- **Reference client:** `clients/faith-chapel.tokens.json`
- **Skills:** 20 foundation + lifecycle skills
- **Index:** `INDEX.md`

Church profiles link via `brand.tokens_file` (see Faith Chapel).

```bash
npm run skills:update-registry   # includes brand-system in manifest.json
```
