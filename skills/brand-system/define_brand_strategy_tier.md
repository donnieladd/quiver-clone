# Define brand strategy tier (Tier 0)

## Purpose

Capture mission, vision, values, audiences before tokens.

## Steps

1. Document mission and vision in one paragraph each.
2. List 3–7 values as verbs or nouns.
3. Define 2–4 primary audiences with needs.
4. Write positioning statement (for X who Y, we Z).
5. Get leadership sign-off before Tier 1.
