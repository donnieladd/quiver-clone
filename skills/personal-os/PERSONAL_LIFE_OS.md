# Personal Life OS

**This is not Faith Chapel. This is how your brain works.**

The document architecture you built at Experience Division was never really *about* a church — it was you learning to **unblur chaos**: separate alignment from approval, execution from governance, feelings from scorecards.

Brand Studio now treats that as your **universal operating system**. Church, corporate, fashion, and creator packs are *applications*. **Personal life is the home base.**

## The struggle (named plainly)

| Work brain | Personal life today |
|------------|---------------------|
| Every ask has a document type | Everything is a vague "I should…" |
| Brief before production | Start mid-stream, burn out |
| Charter before big days | Big life stuff has no sponsor or scope |
| AAR after events | Repeat the same mistakes |
| Scorecard | Guilt instead of data |
| SOP | Reinvent Monday every Monday |

You're not broken. **The environment changed.** Work had structure; home doesn't — unless you install it.

## Same 8 tiers — personal translation

### I. Alignment (Sunday night / before any new thing)

| Doc | Personal use | Example |
|-----|--------------|---------|
| **Creative brief** | Before a project, trip, content batch | "What am I making and for whom?" |
| **Project brief** | Cross-life admin (move, taxes, wedding) | Scope + deadline + who helps |
| **Concept note** | Idea not ready for commitment | "Maybe learn X" — park it, don't chase |
| **Strategy brief** | Quarter or season direction | Q3: health + one creative ship |
| **Vision doc** | Who you're becoming (not mood) | 3-year north star, 1 page |

**Rule:** If you're overwhelmed, you're probably skipping Tier I.

### II. Decision (when money, time, or identity is on the line)

| Doc | Personal use |
|-----|--------------|
| **Proposal** | "Should I take this job/client/trip?" |
| **Decision memo** | Record the call so Future You stops re-debating |
| **Change request** | Life scope creep — "I said yes to too much" |

### III. Governance (ADHD lifelines)

| Doc | Personal use |
|-----|--------------|
| **SOP** | Meds, morning, bills, gym bag — **same steps, no decisions** |
| **SLA** | Response time to family, email, DMs |
| **RACI** | Who owns what at home (even if household of 1: you vs outsourced) |
| **Risk register** | Known personal failure modes (hyperfocus trap, no food, isolation) |

### IV. Planning

| Doc | Personal use |
|-----|--------------|
| **Project charter** | Big life projects: move, launch, health protocol |
| **Run of show** | Travel day, shoot day, important conversation |
| **Playbook** | "How I run my week" — your personal division manual |

### V. Reporting (truth, not shame)

| Doc | Personal use |
|-----|--------------|
| **Scorecard** | 5 lead + 5 lag indicators you actually track |
| **Status report** | Weekly: done / stuck / next |
| **After action report** | After hard weeks, launches, trips — what to keep |

## ADHD-friendly personal stack (minimum viable)

You don't need 40 documents. You need **five rhythms**:

```
1. Weekly alignment brief     (15 min, Sunday night ← locked)
2. Personal scorecard         (5 numbers, not 50)
3. SOPs for non-negotiables   (meds, sleep anchor, money day)
4. Charter for big stuff only (≥4 hours or ≥$500 or high emotion)
5. AAR after big weeks        (20 min, blameless)
```

**Your anchor:** **Sunday night**, ~15 minutes, workflow `personal-os.weekly_alignment`.  
Profile: `personal-os/profile.json`
